Reset Password

The unit will generate a random code and will set LED strip colors using an associated pattern. The code is made up of the first lowercase letter of each of the available colors in the order they are displayed.

Example:
[r]ed, [g]reen, [b]lue, [c]yan, [m]agenta, [y]ellow, [o]range and [w]hite. Which means the challenge code is: rgbcmy

Your password must not contain spaces or emoji.
